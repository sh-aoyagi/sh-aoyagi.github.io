# 三冠馬基準値ファイル（FIREスコア100点の基準）

## 概要
FIREスコアは歴代三冠馬8頭を「100点満点の基準」として算出する。
三冠馬がそのレース・コース・距離に完全適合した場合＝100点。
出走馬はこの基準と比較してパーセンテージで採点する。

---

## 配点ルール（再確認）
| 軸 | 配点 |
|----|------|
| 血統 | 40点 |
| 戦績（経験） | 20点 |
| 仕上がり | 18点 |
| 枠 | 15点 |
| 騎手 | 5点 |
| 天気 | 2点 |
| **合計** | **100点** |

---

## 三冠馬8頭の基準プロフィール

### 1. セントライト（1941年）
- **父系**：プリメロ系（英国血統）
- **得意距離**：1800m〜3000m（中〜長距離）
- **得意コース**：右回り芝
- **ペース適性**：スタミナ勝負・スローからの持続力
- **特記**：時代が古く現代血統との直接比較は困難。基準としてはスタミナ軸で参照。

### 2. シンザン（1964年）
- **父系**：ヒンドスタン系
- **得意距離**：1600m〜3000m（万能型）
- **得意コース**：芝全般・右回り得意
- **ペース適性**：どのペースでも対応（万能）
- **特記**：五冠馬。距離・コースを問わない適応力が最大の特徴。万能型の基準として参照。

### 3. ミスターシービー（1983年）
- **父系**：トウショウボーイ系（ナスルーラ系）
- **得意距離**：2000m〜3000m（中〜長距離）
- **得意コース**：芝・追い込みが決まるコース（直線長め）
- **ペース適性**：ハイペース◎（追い込み型・前半スローは苦手）
- **特記**：極端な追い込み馬。ペース読みが鍵。ハイペース適性の基準として参照。

### 4. シンボリルドルフ（1984年）
- **父系**：パーソロン系（英国血統）
- **得意距離**：1800m〜2500m（中距離万能）
- **得意コース**：芝全般・右回り
- **ペース適性**：どのペースでも安定（精神力・安定感）
- **特記**：七冠馬。国内無敗。安定感・精神力の基準として参照。

### 5. ナリタブライアン（1994年）
- **父系**：ブライアンズタイム系（ロベルト系）
- **得意距離**：2000m〜3000m（中〜長距離）
- **得意コース**：芝・右回り（阪神・中山）
- **ペース適性**：どのペースでも対応・瞬発力＋スタミナ両立
- **特記**：圧倒的な強さ。怖さ知らずの精神力。パワー系血統の基準として参照。

### 6. ディープインパクト（2005年）
- **父系**：サンデーサイレンス系
- **得意距離**：1600m〜2500m（マイル〜中長距離）
- **得意コース**：芝・右回り得意（京都◎）・左回りも対応（東京◎）
- **ペース適性**：スロー〜ミドル◎（後半3ハロンの瞬発力勝負）
- **特記**：現代の種牡馬として最も影響力大。産駒評価の基準として最重要。宝塚記念（阪神2200m）は苦手傾向注意。

### 7. オルフェーヴル（2011年）
- **父系**：サンデーサイレンス系（ステイゴールド）
- **得意距離**：2000m〜3200m（中〜長距離）
- **得意コース**：芝・右回り（阪神・中山・京都）
- **ペース適性**：ハイペース〜ミドル◎（パワー＋スタミナ）
- **特記**：凱旋門賞2年連続2着。パワー型スタミナの権化。長距離・道悪の基準として参照。

### 8. コントレイル（2020年）
- **父系**：サンデーサイレンス系（ディープインパクト）
- **得意距離**：2000m〜2400m（中距離）
- **得意コース**：芝・左回り（東京◎）・右回りも対応
- **ペース適性**：ミドル〜スロー（瞬発力型・父ディープに近い）
- **特記**：父子無敗三冠。現代型中距離の基準として参照。

## レース別・三冠馬対応表

採点時は「このレースのコース・距離に最も近い三冠馬」を基準として使う。

| レース条件 | 参照すべき三冠馬 |
|-----------|----------------|
| 東京芝1600m（安田記念） | ディープインパクト・コントレイル |
| 東京芝2400m（ダービー・JC） | ディープインパクト・コントレイル |
| 阪神芝2200m（宝塚記念） | オルフェーヴル・ナリタブライアン |
| 中山芝2000m（皐月賞・有馬） | シンボリルドルフ・ナリタブライアン |
| 京都芝3000m（菊花賞・天皇賞春） | オルフェーヴル・ディープインパクト |
| 阪神芝1600m（桜花賞・マイルCS） | ディープインパクト・コントレイル |

---

## 採点の手順（Claudeへの指示）

1. 対象レースのコース・距離を確認する
2. 上記「レース別・三冠馬対応表」から参照馬を選ぶ
3. 出走馬の血統・戦績・適性を参照馬と比較する
4. 各軸（血統40点・戦績20点・仕上がり18点・枠15点・騎手5点・天気2点）を採点する
5. 合計点を算出してFIREスコアとする

---

## 注意事項
- セントライト・シンザンは時代が古いため血統の直接比較は困難。スタミナ・万能性の参照に限定する。
- 仕上がり・枠・騎手・天気は土曜日昼（12時）以降に出揃った情報で採点する。
- 情報が出ていない場合は「未確認」として採点を保留し、再サーチする。
