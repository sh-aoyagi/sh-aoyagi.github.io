# 競馬 01_FIREスコア算出.md
# 00_レース特定の結果を受けて実行する

---

## 前提確認

00_レース特定から以下を受け取っていること。

```
・レース名・開催日・競馬場・距離・グレード
・出走馬リスト（全頭分）
・今日書く記事の種類
```

受け取っていない場合は00_レース特定を先に実行する。

---

## FIREスコアの配点（正式版・変更不可）

```
【確定スコア・68点満点】
血統適性   35点
戦績       15点
コース実績 10点
騎手        5点
ローテ      3点

【補足変動・32点満点】
枠              8点
仕上がりタイム  7点
仕上がりコメント 3点
天気            2点
（補足変動は情報が出てから順次加算）

合計100点満点
```

参照：https://note.com/yagiwooo/n/na844da2e1319

---

## STEP1：確定スコアを算出する（68点満点）

出走馬全頭について以下で採点する。

### ① 血統適性（35点）
コース・距離・馬場に合う父母父かを見る。

```
コース・距離ともに適性最高 → 30〜35点
一方が合う               → 20〜29点
不一致                   → 0〜19点

コース別の強い血統：
東京芝2400m：ディープ系・ステゴ系
阪神芝2200m：ステゴ系・ロベルト系・ハーツ系
東京芝1600m：キンカメ系・Storm Bird系
ダート：ルヴァンスレーヴ系・クロフネ系
```

### ② 戦績（15点）
```
G1勝ち：15点
G2勝ち：10点
G3勝ち：7点
未勝利：1点

補正（必ず適用）：
G1初出走：-3点
前走G1大敗（5着以下）：-2点
海外遠征帰り初戦：-1点
前走より400m以上の距離延長：-2点
青葉賞経由のダービー：-3点（追加ペナルティ）
```

### ③ コース実績（10点）
```
同コース同距離での勝利実績：10点
掲示板（〜5着）：5点
初挑戦：0点
```

### ④ 騎手（5点）
```
トップジョッキー：5点
準トップ：3点
その他：1点
```

### ⑤ ローテーション（3点）
```
中4〜8週：3点（最適間隔）
中2週以内：1点
長期休養明け：0点
```

---

## STEP2：補足変動スコアを加算する（情報が出た分のみ）

### ⑥ 枠（8点）※枠順確定後に採点
```
内枠（1〜3枠）：+8点
中枠（4〜6枠）：+4点
外枠（7〜8枠）：-4点
大外（16番以上）：-8点
```

### ⑦ 仕上がりタイム（7点）※土曜12時以降に採点
```
自己ベスト更新：7点
平均的なタイム：4点
平均以下・馬なり重点：2点
```

### ⑧ 仕上がりコメント（3点）※土曜12時以降に採点
「変化の言葉があるか」だけを見る。
```
「良くなった」「シャープになった」「充実」→ +3点
「順調」「いい動き」「問題なし」→ 0点
「リラックス」「のんびり」「感触を確かめた」→ -2点
```

### ⑨ 天気（2点）※当日朝に採点
```
良：2点
稍重：1点
重・不良：0点
```

---

## STEP3：スコアを集計して上位5頭を選出する

```
水曜時点（補足変動なし）：68点満点で暫定スコア
木曜以降（枠確定後）：76点満点
土曜12時以降（全情報揃い次第）：100点満点
```

上位5頭を選ぶ：
```
◎ 本命：【1位】馬名　枠番　騎手　〇点
○ 対抗：【2位】馬名　枠番　騎手　〇点
▲ 単穴：【3位】馬名　枠番　騎手　〇点
△ 要注意：【4位】馬名　枠番　騎手　〇点
× 軽視：馬名　枠番　騎手　〇点
```

### 5枠目のジャッジ
```
スコアが補足変動で75点超えの可能性あり → 「要注目」
スコアが65点以下で変動しても届かない  → 「消し」
```


## STEP5：配信判断

```
【暫定スコア（68点満点・水曜時点）の場合】
グレード G1・G2 → 有料note記事（¥100）として書く
　（補足変動32点を加味すれば75点超えがほぼ確実なため）
　※本文に「枠・仕上がりで最終スコアは変動します」と明記する
　※土曜に補足変動を反映して追記更新する

グレード G3以下 →
　暫定スコア50点以上 → 有料note記事（¥100）として書く
　暫定スコア50点未満 → 今週は保留・CEOに報告

【最終スコア（100点満点・土曜12時以降）の場合】
本命スコア75点以上 → 有料note記事（¥100）＋SNS投稿
本命スコア65〜74点 → 無料SNS投稿のみ
本命スコア64点以下 → 今週は投稿なし・CEOに報告
```
