# 競馬 00_レース特定.md
# 最初に必ず実行する

---

## STEP1：今日の曜日と記事を判断する

今日の日付と曜日を確認して、書く記事を決める。

```
水曜 → 記事③（メイン予想・有料¥100）
木曜 → 記事①（4頭の話A・2頭・無料）
　　　 記事②（4頭の話B・2頭・無料）
金曜16時以降 → 記事④（砂利玉7〜10R・有料）
金曜（時間問わず）→ 記事⑤（歴史・無料）
土曜 → 記事③に最終情報を追記
```

出力：
```
今日は〇月〇日（〇曜日）です。
今日書く記事：記事〇（〇〇）
```

---

## STEP2：必要な情報をCEOに聞く

### 水曜・土曜の場合（メインレース）

CEOにこう聞く：

```
今週のメインレースの出走馬情報を貼ってください。
1レース分をチャットに貼るか、
テキストファイルをアップロードしてください。

欲しい情報：
・レース名・開催日・競馬場・距離・グレード
・出走馬全頭の枠番・馬番・馬名・騎手
```

### 木曜の場合（4頭の話）

CEOにこう聞く：

```
今週のメイン予想で選んだ注目4頭を教えてください。
馬名だけでOKです。
```

### 金曜16時以降の場合（砂利玉7〜10R）

CEOにこう聞く：

```
今週の7〜10Rの出走馬情報を貼ってください。
4レース分をチャットに貼るか、
テキストファイルをアップロードしてください。

欲しい情報（各レースごとに）：
・レース番号・クラス・距離・芝orダート
・出走馬全頭の枠番・馬番・馬名・騎手

※まだ公開されていないレースは「未公開」と書いてください。
　未公開レースはFIREスコアを保留して進めます。
```

### 金曜（歴史記事）の場合

CEOにこう聞く：

```
今週のメインレース名を確認します。
合ってますか？→「〇〇（レース名）」
```

---

## STEP3：情報を受け取って確認する

CEOから情報が届いたら以下を確認する。

```
□ レース名・開催日が今週のものか
□ 競馬場・距離が記載されているか
□ 出走馬の馬名・騎手が記載されているか
□ 「未公開」のレースがあるか
```

未公開のレースがある場合：
```
「〇Rは未公開のため本命印を保留します。
　公開後に追記してください。」
と明記してSTEP4へ進む。
```

---

## STEP4：01_FIREスコア算出.mdへ渡す

以下の形式で情報をまとめて次へ進む。

```
【レース情報】
レース名：
開催日：
競馬場：
距離・コース：
グレード：
今日書く記事：

【出走馬リスト】
枠番 馬番 馬名 騎手
（全頭分）

【未公開レース】
（あれば記載・なければ「なし」）
```
