あなたは顧客インサイト発掘の専門家です。以下のペルソナが抱える「ペイン（悩み・不満・恐怖）」を具体的に掘り下げてください。

## ペルソナ一覧

### A: 公営競技ユーザー
- 競馬好きの30〜50代男性（副業・趣味で参加）
- 週末だけ賭けるライト層〜毎日チェックするヘビー層
- 無料予想では勝てず、有料情報を探している

### B: ダイエッターユーザー
- 忙しい20〜40代男女（会社員・パート）
- コンビニ・冷凍弁当に頼りがちで栄養バランスが不安
- ダイエットを始めては挫折を繰り返している

## 出力形式（JSON）
```json
{
  "personas": {
    "racing_user": {
      "functional_pains": ["〜できない", "〜がわからない"],
      "emotional_pains": ["〜が怖い", "〜が恥ずかしい"],
      "social_pains": ["周りに〜と思われたくない"],
      "jobs_to_be_done": ["本当は〜したい"],
      "trigger_phrases": ["こんな言葉に反応する", "刺さるコピー例"]
    },
    "diet_user": {
      "functional_pains": [],
      "emotional_pains": [],
      "social_pains": [],
      "jobs_to_be_done": [],
      "trigger_phrases": []
    }
  }
}
```

各ペインは具体的・感情的な言葉で5〜8項目書いてください。最後に「このペルソナが思わず買ってしまうコンテンツ条件」を各ペルソナ50字以内でまとめてください。
