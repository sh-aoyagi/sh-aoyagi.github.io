# 自キュン 04_state更新.md
# ブロック生成後に必ず実行する

---

## 前提確認

以下を読んでから更新する。

```
① 今生成したブロックの本文
② 現在のstory_state.md
```

---

## STEP1：現在地を更新する

```
今のブロックで何が起きたかを一行で書く。
例：「Block3：教授が藤村のアドレスを中村に渡したことが判明」
例：「Block4：藤村が迷惑メールの山から中村のメールを発見」
```

---

## STEP2：感情値を更新する

```
現在の感情値を読む。
今のブロックの出来事から増減を判断する。

目安：
・フィジカルが止まった瞬間　→ +10〜15
・相手の行動に気づいた　　　→ +5〜10
・第三者が動いた　　　　　　→ +5
・何もしないを選んだ　　　　→ -5〜10
・迷っている　　　　　　　　→ ±0〜5

感情値100に近づくほど着地に近い。
```

---

## STEP3：伏線を更新する

```
【新しく発生した伏線】
このブロックで置いた伏線を追加する。
例：「封筒が止まった理由」
例：「PHSと嘘をついた理由」

【回収済み伏線】
このブロックで回収した伏線を移動する。
例：「封筒＝応援」→ 回収済みに移動
```

---

## STEP4：禁止事項を確認・追加する

```
既存の禁止事項を確認する。
このブロックで新たに発生した禁止事項があれば追加する。
例：「中村がLINEを知っていることは最後まで明かさない」
```

---

## STEP4.5：スプシに保存する

生成したブロック本文をスプレッドシートに書き込む。

```
スプレッドシートID：1ni70t6Er3Ri2s_mU2OR-5K6e0vwO9iyj
シート名：自キュン原稿
書き込み先：該当話の列・該当ブロックの行
```

書き込み後「保存しました」と報告する。

---
