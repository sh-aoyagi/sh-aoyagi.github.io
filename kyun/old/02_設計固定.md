# 自キュン 02_設計固定.md
# 01_企画生成のOKが出たら実行する
# ここで決めたことは話が終わるまで変更禁止

---

## 前提確認

01_企画生成から以下を受け取っていること。

```
・時代・場所・季節
・主人公・相手・第三者
・テーマ
・あらすじ
・ラスト候補（①②③）
```

---

## STEP1：登場人物を固定する

```
主人公：
　名前（苗字のみ）：
　職業・年齢：
　欠けているもの：
　隠れた属性（0〜2個）：
　行動タイプ：躊躇型 / 猪突猛進型 / 気づかない型 / 諦め型

相手：
　名前（苗字のみ）：
　職業・年齢：
　特徴（不器用・直接言わない・言葉より行動）：

第三者：
　名前：
　タイプ：中村型 / マスター型 / 自販機型 / 砂川型
　役割（一言で）：
```

**砂川型の定義：**
男を通す・二人の邪魔をしない・相手の良さをわかっている・格好いい引き立て役

**全タイプ共通：誰も悪い人にしない**

---

## STEP2：テーマと禁止事項を固定する

```
テーマ（主人公が乗り越えること）：

禁止事項：
・告白で解決しない
・テーマを忘れない
・相手は直接言わない
・第三者は答えを出さない（時間か背中を与えるだけ）
・（話ごとに追加する禁止事項）：
```

---

## STEP3：モチーフと道具を固定する

```
メインモチーフ（道具）：
道具の非対称（知っている側・知らない側）：
道具が言い訳の装置として機能する場面：
道具が役割を終える瞬間（＝二人の時間が終わる瞬間）：
```

---

## STEP4：構成を固定する

```
有料パートの切れ目（最大の緊張感の瞬間）：
視点変更：あり（◆で明示）/ なし
時間の飛躍：あり（　年後）/ なし

【時間飛躍ルール】
・飛躍先は必ず現在（執筆時点）= 2026年
・PCの時計から未来を書くことは禁止
・時代設定の下限：1980年（昭和55年）以降
・昭和設定は1980年〜1989年の範囲のみ
・飛躍後の描写は「2026年の今も〜」か逆算して年数を決める
　例）1996年の話 → 「三十年後」= 2026年
主人公の判断：正しい / 馬鹿な
ラスト確定（①②③から選ぶ）：
```

---

## STEP5：感情値を設定する

```
感情値初期値：　／100
現在地（話の冒頭の状況）：
```
