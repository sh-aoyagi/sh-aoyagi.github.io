# 自キュン 07_マージ.md
# 06_校正完了後に実行する

---

## 絶対ルール

```
文章を書き換えない
語尾を書き換えない
描写を追加しない
削除しない
接続詞だけ補完可
目的は編集ではなく結合作業
矛盾があっても勝手に直さない
```

---

## ファイル構成

```
C:\denno\kyun\episodes\XX話\
├ block01.txt
├ block02.txt
├ block03.txt
...
└ story_state.md
```

---

## STEP1：ブロックを順番に読む

全ブロックを一度に読まない。
隣り合う2ブロックずつ確認する。

```
Block1の末尾 ↔ Block2の冒頭
Block2の末尾 ↔ Block3の冒頭
...を繰り返す
```

---

## STEP2：接続の判断

各ブロックの境界で以下を判断する。

```
【場転・時間経過あり】
→ 改行を入れる（「———」で区切る）
→ 接続詞は不要

【場転・時間経過なし（同じ場面の継続）】
→ そのまま並べるだけ
→ 接続詞も不要

【会話のラリーが続く場合】
→ そのまま並べるだけ
→ セリフは切らない
```

---

## STEP3：接続詞の補完

境界が不自然な場合のみ補完する。

```
補完できるもの：
・改行
・「———」（場面転換）
・「それから」「その後」などの最小限の接続詞

補完できないもの：
・新しい描写
・感情の説明
・セリフ
・場面の追加
```

---

## STEP4：結合して出力する

全ブロックを結合した完成原稿を出力する。

```
保存先：C:\denno\kyun\episodes\XX話\完成原稿.txt
```

---

## STEP5：マージ後チェック

結合後に以下だけ確認する。

```
・ブロックの順番が正しいか
・有料パートの🔒が正しい位置にあるか
・最後の一文が1行独立しているか
```

問題があればCEOに報告する。
勝手に修正しない。

---

## 注意

マージが破綻した場合：
1ブロック1ファイルのまま真平さんに渡す。
「結合できませんでした」と正直に報告する。
無理に結合しない。
