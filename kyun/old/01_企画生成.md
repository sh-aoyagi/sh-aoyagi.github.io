# 自キュン 01_企画生成.md
# 00_調査の結果を受けて実行する

---

## 前提確認

00_調査から以下を受け取っていること。

```
・人気傾向
・禁止リスト（時代・場所・関係性・道具）
・時代の匂い候補
```

受け取っていない場合は00_調査を先に実行する。

---

## STEP1：設定ガチャ

禁止リストに含まれない組み合わせで設定を生成する。
人気傾向に近い設定を優先する。

```
時代：昭和 / 平成 / 令和（禁止リストを除く）
場所：（禁止リストを除く）
季節：春 / 夏 / 秋 / 冬
主人公：職業・年齢・何か一つ欠けているもの
相手：不器用・直接言わない・言葉より行動
第三者：中村型 / マスター型 / 自販機型
```

時代の匂い候補を設定に1つ入れる。

出力：
```
時代：
場所：
季節：
主人公：
相手：
第三者：
時代の匂い：
```

---

## STEP2：あらすじガチャ

100文字以内で生成する。

ルール：
```
・恋愛は結果であってテーマではない
・主人公が行動することで未来が変わる
・何もしなければ何も起きないまま
・主人公は常に女性
```

出力：
```
あらすじ（100文字以内）：
```

---

## STEP3：テーマ設計

主人公が乗り越えることを一言で決める。

```
例：「待つことをやめる」
例：「完璧じゃなくていいと気づく」
例：「バイアスより本能を信じる」
```

出力：
```
テーマ：
```

---

## STEP4：ラストの一文候補

3案生成する。

ルール：
```
・10文字以内
・道具か行動で終わる
・感情の名前を使わない
・倫理に反する行動を含まない
```

出力：
```
ラスト候補①：
ラスト候補②：
ラスト候補③：
```

---

## STEP5：次フェーズへ渡す

以下を確認してOKが出たら02_設計固定.mdへ渡す。

```
【企画サマリー】
時代・場所・季節：
主人公：
相手：
第三者：
テーマ：
あらすじ：
ラスト候補：①②③
```

CEOに確認を取る。OKが出るまで次に進まない。
