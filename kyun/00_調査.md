# 自キュン 00_調査.md
# 企画生成の前に必ず実行する

---

## STEP1：集計データを読む

スプレッドシートの「NOTE集計」シートを読む。
URL：https://docs.google.com/spreadsheets/d/1ni70t6Er3Ri2s_mU2OR-5K6e0vwO9iyj/

**スプシが読めない場合：**
```
STEP1・STEP2をスキップして
STEP3（トレンド調査）から始める。
禁止リストは空欄のまま01_企画生成.mdに渡す。
```

自キュンシリーズの記事のみ抽出して以下を確認する。

```
確認項目：
- ビュー数上位3話
- スキ数上位3話
- 直近3話のタイトル・設定（時代・場所・関係性・道具）
```

出力形式：
```
【人気上位】
1位：XX話「タイトル」ビューXX スキXX
2位：XX話「タイトル」ビューXX スキXX
3位：XX話「タイトル」ビューXX スキXX

【人気の傾向】
（時代・場所・関係性・道具の共通点を一言で）

【直近3話の設定】
N話：時代＿　場所＿　関係性＿　道具＿
N-1話：時代＿　場所＿　関係性＿　道具＿
N-2話：時代＿　場所＿　関係性＿　道具＿
```

---

## STEP2：重複禁止チェック

直近3話で使った設定を「禁止リスト」に入れる。

```
禁止リスト（次の話で使えない）：
- 時代：＿＿＿
- 場所：＿＿＿
- 関係性：＿＿＿
- 道具：＿＿＿
```

3話より前なら再利用OK。

---

## STEP3：トレンド調査

以下をWebSearchで調べる。

```
検索1：「今 流行 女性 SNS（月・年）」
検索2：「（現在の季節）トレンド 恋愛 note」
```

時代の匂いになりそうなものを1〜3個抽出する。

例：ラブブ流行中 / 日経6.5万 / AI副業ブーム / 推し活文化

---

## STEP4：ターゲット確認（固定）

```
主人公：女性・固定（変更不可）
ターゲット：腐女子・BL好き女性・感情移入型読者
テーマの核：行動した人だけが未来を変えられる
　└ 行動する → 恋人になる・昇進する・再会する
　└ 何もしない → 何も起きないまま
```

---

## STEP5：次フェーズへ渡す

以下を`01_企画生成.md`に渡す。

```
【調査結果サマリー】
人気傾向：＿＿＿
禁止リスト：時代＿ / 場所＿ / 関係性＿ / 道具＿
時代の匂い候補：＿＿＿
```
