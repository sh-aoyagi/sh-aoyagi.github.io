# 自キュン 06_校正.md
# 05_品質チェックの結果を受けて実行する

---

## 絶対ルール

```
内容変更禁止
削除禁止
追加禁止
以下の4種類のみ修正可能：
　① てにをは
　② 誤字
　③ 重複表現
　④ 連続パターンの解消
```

---

## 修正①：てにをは

助詞の誤りを修正する。内容は変えない。

---

## 修正②：誤字

明らかな誤字のみ修正する。
表記ゆれ（ひらがな・カタカナ）は原則触らない。

---

## 修正③：重複表現

同じ言葉が近い場所に重複している場合のみ修正する。
例：「扉を開けて、扉の向こうに」→「扉を開けて、その向こうに」

---

## 修正④：連続パターンの解消

以下のパターンが3回以上続いている場合のみ修正する。

**文末の連続：**
```
～た。～た。～た。が続く場合：
3つ目の文末を以下のどれかに変える
・体言止め（名詞で終わる）
・「～る」現在形
・「～だ」
・短文1語

例）
修正前：扉を開けた。声が聞こえた。息が止まった。
修正後：扉を開けた。声が聞こえた。息が、止まる。
　または：扉を開けた。声が聞こえた。静止。
```

**描写の連続（多角列挙への変換）：**
```
情景・情景・情景が続く場合：
3つを多角列挙に変換する

多角列挙のルール：
・切り口を別々にする
　（色・擬人化・感情の距離・時間・温度・音など）
・文字数を揃えない（短・中・長 または 中・短・長）
・同じ文字数を2つ以上続けない

例）
修正前：赤いポスト、夕焼け、彼の後ろ姿
修正後：赤いポスト（色・6文字）
　　　　寂しそうな夕焼け（擬人化・9文字）
　　　　手の届かない彼の後ろ姿（感情の距離・13文字）

セリフの連続はNG対象外。会話のラリーはそのままでよい。
```

## 注意

修正は最小限にとどめる。
迷ったら修正しない。
「直した方がいい気がする」は修正理由にならない。
