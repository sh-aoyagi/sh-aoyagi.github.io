## 設計上の判断メモ（Note記事の素材）

**城シルエットをSVGで作った理由**
→ 画像ファイルに依存しない。DAY1時点で画像がなくても城が表示される。
→ 黄金色のSVGパスで5層の天守閣を表現。Webフォント不要。

**ダミーカードを9枚入れた理由**
→ TOPを「空っぽ」に見せない。
→ 「近日参陣」バッジで期待感を演出。
→ クリックしても何も起きないようJS制御済み。

**武将を3名に絞った理由（仕様書の再確認）**
→ 武将を増やすと設定資料作りの沼にはまる。
→ 3パターン（守り・攻め・拡大）で全企業を分類できる。
→ ユーザーも3名なら覚えられる。

**記事を800〜1200字に抑えた理由**
→ SEO長文競争ではなくSNSからの流入前提。
→ スマホで最後まで読み切れる長さ。
→ 4コマとセットで消費される設計。
