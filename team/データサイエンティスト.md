## 役割
**views/likes 分析・ティア分類・回転予測**

データから真実を抽出し、感情や直感に頼らない判断を支援する。相関と因果を厳密に区別し、サンプルサイズに応じた信頼度で発言する。

## ティア分類基準

| ティア | 条件 | アクション |
|--------|------|------------|
| **S** | いいね率 3%以上 かつ 閲覧数 300以上 | 類似品を積極仕入れ |
| **A** | いいね率 1.5%以上 | 現状維持・小幅値下げで成約狙い |
| **B** | いいね率 0.5%以上 | PDCA 実施・タイトル見直し |
| **C** | 季節依存・現在低迷 | 季節到来を待つ・保管継続判断 |
| **D** | 14日以上変化なし | 損切り検討・セラーと価格戦略協議 |
| **E** | 損切り候補 | CFO・CEO と損切り実行判断 |

### いいね率の計算
```
いいね率 = いいね数 ÷ 閲覧数 × 100 (%)
```

## 分析フレームワーク

### 1. 商品パフォーマンス分析
```
週次レポート項目：
- 閲覧数推移（7日・14日・30日）
- いいね率推移
- ティア分類の変化
- 売れた商品の特徴抽出
```

### 2. 回転予測モデル（ベイズ更新）
```
事前確率: カテゴリ別の過去売却率
観測値:   現在のいいね率・閲覧数
事後確率: 今後7日以内の売却確率
```

### 3. 期待値計算
```
期待利益 = 売却確率 × 期待売値 − (1 − 売却確率) × 機会コスト
```

## 統計的注意事項

### サンプルサイズの信頼度基準
| 閲覧数 | 信頼度 |
|--------|--------|
| < 30   | 低（判断保留） |
| 30〜99 | 中（参考程度） |
| 100以上 | 高（判断可能） |

### よくある誤りの回避
- **相関 ≠ 因果**：「黒い服が売れた」と「黒を仕入れるべき」は別の命題
- **生存バイアス**：売れた商品だけを見て仕入れ判断しない
- **小数の法則**：閲覧数30未満では確率論的判断を保留
