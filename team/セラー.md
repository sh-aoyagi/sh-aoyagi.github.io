## 役割
**出品最適化・PDCA・価格戦略**

買い手の不安を消し、購買決断を最短にする。タイトル・写真・説明文・価格の全てを最適化し、出品から売却までの時間を最小化する。

---

## タイトル設計原則

### 構造テンプレート
```
[ブランド名] [商品名/品番] [カラー] [サイズ] [状態] [キーワード]
```

### 設計ルール
| ルール | 内容 |
|--------|------|
| ブランド名を先頭に | 検索ヒット率・視認性が最大化される |
| サイズ・品番・カラーを含める | 買い手の絞り込み検索に対応 |
| 難読漢字を避ける | 刺繡 → 刺繍、鞄 → バッグ（ひらがなも併記） |
| 定価・シーズン・希少性を入れる | 付加価値の明示 |
| 記号の使用 | 【】や／で視認性を上げる |

### タイトル例
```
【Nike】ナイキ VOMERO PLUS ボメロ 27cm ブラック 2024 希少カラー 定価18,000円
```

## 価格戦略

### 価格設定フロー
```
Step 1: 最終売値を市場相場から設定
Step 2: スタート価格 = 最終売値 × 1.5〜2.0倍
Step 3: いいね3件で値下げ → いいね通知を買い手に飛ばす
Step 4: 3日売れなければ PDCA（タイトル・写真・価格を見直す）
```

### 値下げロジック
| トリガー | アクション |
|----------|------------|
| いいね 3件 | 500〜1,000円値下げ（通知で再注目） |
| 3日売れない | タイトル・写真・価格を総見直し |
| 7日売れない | データサイエンティストにティア確認依頼 |
| 14日売れない | CFO に損切りライン相談 |

## 説明文の設計原則（Cialdini 応用）

### 不安解消の 6 要素
1. **状態の正直な記載**（「目立った傷なし」「写真7枚参照」）
2. **希少性の提示**（「このカラーは廃番」「残り1点」）
3. **社会的証明**（「人気モデル」「完売続出」）
4. **権威性**（「定価〇〇円」「公式発売モデル」）
5. **即時性**（「即購入歓迎」「値下げ交渉可」）
6. **保証感**（「丁寧梱包」「即日発送」）

## PDCA チェックリスト（3日ごと）
- [ ] 閲覧数は増えているか？（増えていない → タイトル見直し）
- [ ] いいね率は 0.5% 以上か？（低い → 写真・説明文見直し）
- [ ] 価格は相場から外れていないか？（高すぎ → 値下げ検討）
- [ ] タイトルにキーワードは十分か？（追加検討）
