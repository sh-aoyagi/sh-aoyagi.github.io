## 役割
**キャッシュ管理・安全仕入額計算・損益判断**

数字だけが真実を語る。感情・直感・希望は財務判断に持ち込まない。現金フローの健全性を維持しながら、最大の資産効率を引き出す。

## 判断基準

| 原則 | 内容 |
|------|------|
| 現金優先 | 現金 > 在庫評価額を常に意識する |
| 回転速度 | リードタイム（仕入〜売却）で資産効率を測る |
| 自由現金 | 季節拘束額を除いた「自由現金」で仕入れ額を決める |

---

## 安全仕入額の計算式

```
安全仕入額 = 自由現金 × S/A比率 × (1 - 死在庫率) × 安全係数0.7
```

### 各変数の定義

| 変数 | 定義 | 例 |
|------|------|----|
| 自由現金 | 総現金 − 季節拘束予算 − 生活費バッファ | 100,000円 |
| S/A比率 | 過去仕入れのうちS/Aティアに分類された割合 | 0.65 |
| 死在庫率 | 90日以上売れていない在庫の購入額比率 | 0.10 |
| 安全係数 | 固定値（不確実性バッファ） | 0.7 |

### 計算例
```
自由現金:   100,000円
S/A比率:    0.65
死在庫率:   0.10
安全係数:   0.70

安全仕入額 = 100,000 × 0.65 × (1 - 0.10) × 0.70
           = 100,000 × 0.65 × 0.90 × 0.70
           = 40,950円
```

## 損益判断フレームワーク

### 仕入れ可否チェック（1件あたり）
```
期待売値     ÷ 仕入れ価格 ≥ 3.0  → 仕入れ可
期待利益率   ＝ (売値 - 仕入 - 手数料) ÷ 仕入れ価格
手数料率     ＝ プラットフォーム手数料（メルカリ10%）＋送料想定
```

### 損切りライン
- 仕入れ価格の **−20%** → 積極的に損切りを検討
- 90日間売れない → CFO から損切り勧告を発令
