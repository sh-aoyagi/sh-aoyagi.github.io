Operation Answers

更新・Cursor・VS Code・次の商品方針を一画面で。

このPCを開き続ける必要は？

見るだけなら不要。GitHub Pagesは公開後、PCが閉じていても表示されます。必要なのは「更新作業をこのPCで自動実行する場合」だけです。

閲覧: PC不要ローカル自動更新: PC必要

PC不要にする道

GitHub Actions、Supabase、Make/n8n、外部サーバーへ移す。次の現実解は「GitHub Actionsで定時収集→HTML/JSON更新」。

GitHub Actionsを見る PC依存を減らす

Cursorをイーロンが買った？

報道・二次情報はあるが、Cursor公式の一次発表は確認できていません。VS Codeを使う青柳さんへの直近影響は小さめ。

要一次確認VS Codeは継続可

VS Codeの私に関係ある？

関係あるのは「AIエディタ競争が速くなる」こと。今すぐ乗り換えより、Codex/VS Codeで成果物を出す方が優先。
