# 競馬予想note記事生成ルール
# プロジェクト：02-noteマネタイズPJT-競馬予想編

---

## 1. コンセプト

「FIREするための競馬予想記事」。
競馬で勝ち続けて経済的独立（FIRE）を目指す読者に向けた、
歴史（血統・過去成績）と今（仕上がり・追い切り情報）を掛け合わせた予想コンテンツ。
配信は**レース3日前**が基本。この「直前仕上がりを反映できる」タイミングが差別化ポイント。

---

## 2. 独自スコアリング名

**FIREスコア（ファイヤースコア）**
正式名：**FIRE = Final Insight Racing Evaluation**

### 使い方の文例
- 「独自スコアリング『FIRE（ファイヤー）』によると〜」
- 「FIREスコア78——激走候補」
- 「このFIREスコアが示す条件が今週揃った」

### スコア基準（目安）
| スコア | 評価 |
|--------|------|
| 75以上 | 激走候補 |
| 65〜74 | 有力 |
| 64以下 | 参考 |

### スコアの構成要素（記事内で説明する場合）
- 血統適性（このコース・距離・ペースに合う血か）
- 過去成績パターン（歴史的に同条件でどう走ったか）
- 直前仕上がり（追い切りタイム・調教内容）
- 展開一致度（想定される展開でこの馬の脚が活きるか）

## 3. リーガルチェック：必須ルール

### 3-1. ウマ娘（Cygames）関連【最重要】
- **有料記事・収益化コンテンツに「ウマ娘」という言葉・キャラ名（ゲームキャラとして）を使用禁止**
- 根拠：Cygames二次創作ガイドライン 2025年4月28日改定
  「商業利用または営利を目的とするもの（同人活動の範囲を超えるもの）はご遠慮ください」
- **実馬の名前（ゴールドシップ・オルフェーヴル等）は問題なし。**
  ゲームキャラとして言及・想起させる文脈がNG。
- タグに「ウマ娘」を使うことも有料記事ではNG
- カバー画像にウマ娘キャラを想起させる生成AI画像を使うのもNG

**NG例：**
「ウマ娘『ゴールドシップ』の甥っ子にあたるジュウリョクピエロ」

**OK例：**
「名馬ゴールドシップ（G1・6勝）と同じステイゴールドの血を引くジュウリョクピエロ」

---

### 3-2. JRAコンテンツ関連
- JRA公式の映像・画像・記事文章の転載・二次利用は禁止（私的利用を超える場合）
- 動画・映像へのリンクも営利・非営利問わずNG（JRA HP利用規約明記）
- **レース名・距離・競馬場・日程・馬名・血統情報・オッズ・タイム等の事実情報は著作権対象外でOK**
- JRA-VAN のデータを引用する場合は「JRA-VAN投稿ガイドライン（令和7年改定）」の範囲内で
  - 広告収益つきYouTube・note有料記事でもガイドライン範囲内なら利用可
  - JV-Link取得データの投稿は不可
  - 画面キャプチャは「一部として写り込む」程度ならOK、独自データだけ切り出してまとめるのはNG
- 参照URL：https://jra-van.jp/info/post_guide.html

---

### 3-3. 予想販売行為全般
- 「必ず当たる」「絶対」「確実に」等の断定表現は禁止（詐欺罪リスク）
- 馬券共同購入・資金集めの呼びかけは禁止（出資法違反リスク）
- 独自スコア（FIREスコア）は「根拠ある特殊情報」と誤認させないよう説明を添える

展開分析・エンターテインメントコンテンツです。
的中・勝馬を保証するものではありません。
馬券の購入は必ずご自身の判断と責任のもとで行ってください。
```

### 基本構成
```

販売ページ説明文（免責一行を含む）
タグ
カバー画像メモ
───────────
【無料パート】

  - FIREスコア説明
  - コンセプト（血統・歴史の導線）
  - 注目馬3頭（①有力馬評価 ②展開 ③妙味）

【有料パート】
  - 有力馬評価テーブル（印・スコア・根拠）

## 5. 命名・ブランディングのルール

- スコア名：**FIREスコア（FIRE = Final Insight Racing Evaluation）**
- スコア説明は有料パートの冒頭に必ず入れる
- ウマ娘との連想はコンテンツの方向性として意識してよいが、
  **テキスト上でウマ娘・ゲームキャラ名を有料記事に書くことは禁止**
- 実馬の歴史・血統をゲームの「固有スキル発動」に相当するものとして
  読者が楽しめる文体で書く（書き方の工夫でOK）
